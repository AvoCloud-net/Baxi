3.6 Prism Trust System

Prism is Baxi's network-wide trust scoring system. It automatically assigns each user a score between 0 and 100 based on moderation events across all servers running Baxi. By using any server with Baxi, you acknowledge that:

  • Your moderation history (warnings, kicks, bans, filter violations) contributes to a trust score that is shared across all Baxi servers.
  • Users with a trust score of 30 or below are automatically flagged and blocked from Global Chat.
  • Server administrators can view flagged users via the /scan_users command.
  • Prism only tracks behaviour resulting in moderation actions — normal messages are never scored or stored.
Moderation tool, not profiling

Prism is strictly a moderation tool. It does not profile users, track activity patterns, monitor message content, or build social graphs. The only data Prism records are moderation actions (e.g. warnings, kicks, bans) and the resulting trust score. No behavioural surveillance or identity profiling takes place.

Trust scores recover over time through clean behaviour. You can opt out of Prism at any time by running /prism-optout in any server with Baxi. This is a toggle command — running it once opts you out and deletes your profile, running it again opts you back in. You may also request a copy or deletion of your Prism data by contacting us.

3.7 Global Chat

Global Chat is a cross-server chatroom connecting all participating Baxi servers. The following rules apply:

  • Content Policy: All messages in Global Chat are filtered by Baxi's AI-powered chat filter at all times. Content that violates any server's moderation rules may result in removal and penalties to your Prism trust score.
  • Shared Space: Messages sent in Global Chat are visible to all connected servers. Do not share personal information, NSFW content, or any material that violates Discord's Terms of Service.
  • Access Restrictions: Users with a Prism trust score of 30 or below are automatically blocked from participating in Global Chat.
3.8 AI-Powered Features

Baxi uses AI models (including our self-hosted Llama Guard instance) for content moderation and chat filtering. While we strive for accuracy, automated systems may occasionally produce false positives or miss certain content. Server administrators are responsible for reviewing moderation actions and adjusting filter settings as needed. AI-based decisions do not replace human judgement — they are tools to assist moderation.
