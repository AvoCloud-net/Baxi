    NETWORK SAFETY

    Network Safety

    A shared safety list of users that a human moderator has banned or reported, so participating servers can catch known bad actors early when they try to join.

    When enabled, bans your moderators issue on this server contribute a minimal entry to the network safety list (the user ID + a coarse category like raid/spam/hate, set by the moderator — no message content, no behavior log, no score). Other participating servers can then screen joins against it. Use Mod-Gate to act on the list when users join. Users can opt out via /safety_optout.

    ANTI-RAID

    Anti-Raid

    Baxi learns this server's normal rate of joins and messages over time. When the live rate spikes far above the baseline (a join wave or message flood), the server is automatically locked down: invites paused, verification raised, slowmode applied, and raiders timed out. Everything is restored when the lockdown expires.

    Balanced — sensible default for most servers.

    Where raid lockdown / lift alerts are posted.

    Soft-slows a single channel when it gets flooded with normal chatter — no lockdown, no punishment. The full raid lockdown still handles coordinated attacks.

    Messages within the message window in one channel before it gets slowed.

    How long the soft slowmode stays before lifting automatically.

    Advanced settings

    Higher = trips sooner (more aggressive). Lower = more tolerant of busy periods.

    Minimum joins inside the join window before a raid can be declared, regardless of baseline.

    Minimum messages inside the message window before a message-flood raid can be declared.

    INSTAGRAM TRACKING

    Instagram Tracking

    Get notified when tracked Instagram accounts post new photos or reels. Checks every 15 minutes. Only public accounts are supported.

    Instagram Tracking currently unavailable

    Instagram blocks all automated access from server infrastructure — including authenticated API requests. The official Meta Graph API requires a verified Business Portfolio, App Review, and approved permissions, which are gated behind a review process that Meta does not grant to third-party bots. All known workarounds (scraping, mirror sites, unofficial APIs) are either blocked or unreliable. This feature is preserved in the codebase and will be re-enabled if Meta makes access feasible in future.

    Notifications will be sent here unless overridden per account.
