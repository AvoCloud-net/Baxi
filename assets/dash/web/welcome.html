WHAT BAXI CAN DO

Features

Everything you need to run a great Discord server — configured in minutes from this dashboard.

Moderation

Ban, kick, mute, warn — with temp durations, auto-escalation and DM notifications.

Chat Filter

SafeText, AI-powered Qwen2.5 (qwen2.5:3b via Ollama with a custom moderation system prompt) and phishing URL detection — all in one system.

Anti-Spam

Real-time detection of message floods, mention spam and repeated content.

Welcomer

Custom join/leave messages with images, colors and member variables.

Live Stats Channels

Voice channels that display member count, bots, roles and more — updated automatically.

Livestream Tracker

Twitch go-live notifications with live viewer counts and automatic channel updates.

Global Chat

Cross-server chat rooms with 24/7 moderation and a shared flagged users database.

Temp Voice Channels

Auto-create and delete voice channels when members join a trigger channel.

Support Tickets

Member support system with staff roles, transcripts and a full dashboard log.

Prism — Trust Scoring

Automated behavior analysis. Every user gets a trust score based on violations across all Baxi servers — bad actors get flagged automatically.
